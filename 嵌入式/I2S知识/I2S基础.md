# I2S基础
## 概述
I2S总线又称 集成电路内置音频总线，主要用于音频的一些传输标准

## 特点
通常为4线
|          引脚线          | 备注                                  |
| :----------------------: | ------------------------------------- |
|       时钟线(CLK)        | 用于产生时钟                          |
|    帧时钟线(LRCK/WS)     | "1"表示传输右声道数据,为"0"则是左声道 |
| 串行数据输出线(DATA_OUT) | 数据由高到低输出(16位,24位,32位)      |
| 串行数据输入线(DATA_IN)  | 数据由高到低输入                      |
