# control 寄存器

control 默认全0

| [31:3] | 2   | 1         | 0         |
| ------ | --- | --------- | --------- |
| -      | 略  | **SPSEL** | **nPRIV** |

| SPSEL | `处理模式, 该位为0且写无效` |
| ----- | --------------------------- |
| 0     | 线程模式 使用 **MSP**           |
| 1     | 线程模式 使用 **PSP**           |

| nPRIV         |                                                    |
| ------------- | -------------------------------------------------- |
| 0(特权等级)   | **特  权** `---->` **非特权** (任何情况都行)       |
| 1(非特权等级) | **非特权** `---->` **特  权** (只能在异常处理期间) |

- `PSP` 只能用于线程模式
- `处理模式` 执行中断服务程序(ISR)等异常处理, 在处理模式下, 处理器总具有 `特权访问等级`
- `线程模式` 普通的应用代码时, 处理器可以出于 `特权访问等级` 和 `非特权访问等级`, 具体由 `control` 寄存器控制
