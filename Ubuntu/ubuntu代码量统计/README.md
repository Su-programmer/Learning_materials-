# 代码量统计
> 编写代码的时候经常会遇到代码量统计相关的事情工作, 这儿推荐大家一个代码量统计的工具`cloc`

## 安装`cloc`

```
sudo apt install cloc
```

---

## 执行统计命令

- `cloc [统计的目录]` 如:
```
cloc testproject/
```
