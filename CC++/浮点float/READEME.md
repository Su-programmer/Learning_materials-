# 浮点 float

## 32位 单精度浮点数

| 符号位 |  指数   | 小数位 |
| :----: | :-----: | :----: |
|   31   | 30 - 23 | 22 - 0 |

若 第 n 位表示 x[n]

|       |                                                   |
| :---: | :-----------------------------------------------: |
| 符号  |                  (-1) ^ (x[31])                   |
| 指数  |              2 ^ (x) , x ∈[-127, 0]               |
| 小数  | (1 + (1/2)^(x[22]) + ... + (1 / (2 ^ 23)) ^ x[0]) |

value =符号 x 指数 x 小数
