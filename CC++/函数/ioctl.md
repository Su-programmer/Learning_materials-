# ioctl

```c
int ioctl(int fd, int request, ...);
```

> 系统调用为执行文件和设备操作提供了一种多用途机制

**参数:**

| name      | des        |
| --------- | ---------- |
| `fd`      | 文件描述符 |
| `request` | 控制操作   |
| `...`     | ...        |
