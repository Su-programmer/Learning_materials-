# gitbook 入门

## linux安装

`sudo apt-get install nodejs`

`sudo apt-get install npm`

`sudo npm install gitbook-cli -g`

`sudo apt-get install calibre`

`gitbook pdf ./ ./a.pdf`

`gitbook init`

`gitbook build`

`gitbook serve`
