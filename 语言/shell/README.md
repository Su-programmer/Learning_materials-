# shell 语法

## readonly 只读变量

```shell
str="iotxiaohu"
readonly str    # str 为只读变量
```

## 文件操作符

| 操作符 | 含义                                      |
| ------ | ----------------------------------------- |
| -e     | 判断对象是否存在(Exist)，若存在则结果为真 |
| -d     | 判断对象是否为目录(Directory)，是则为真   |
| -f     | 判断对象是否为一般文件(File)，是则为真    |
| -r     | 判断对象是否有可读(Read)权限，是则为真    |
| -w     | 判断对象是否有可写(Write)权限，是则为真   |
| -x     | 对象是否有可执行(eXcute)权限，是则为真    |

`[ -f a.txt ]`  --- a.txt 文件存在就为真
